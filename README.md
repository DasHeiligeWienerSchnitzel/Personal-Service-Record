# Personal-Service-Record
## Features
- Every player gets their own Personal Service Record (short PSR). 
- For now there are 3 pages containing various information about the player.
- Players can look at other PSRs.
- Is persistent over missions.
- Can create own awards and give them to players.
- Whitelist for admins to allow editing of certain information, like training for example.
### Page 1
- Contains General Information about the player.
 - Name, Rank, Section, Role, ID
- Rank, Section and Role can be edited.
### Page 2
- Contains one the left the side all the possible trainings. There greyed out if not done and full colour otherwise. 
- Can be edited to switch state between done or undone.
- The right side contains various stats that are tracked throughout the missions.
 - Enemies Killed, Distance Walked, Hours played, Sessions played, Deployments completed, Treatments Succeeded, Explosives Defused, Trenches Dug.
- Those stats can not be edited.
### Page 3
- Contains various medals the player has achieved over his playtime.
- Can be removed if necessary.
### Creation of awards.
- New Awards can be created by admins. 
- Icon, Title and Description can be chosen and customised.
### Award Giver
- Awards can then be given to other players.
- Just selected which ones or all and then they get their new award.
